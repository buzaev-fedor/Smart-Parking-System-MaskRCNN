# Smart-Parking-System-MaskRCNN
Программа обработки RTSP потока для анализа свободных и занятых парковочных мест. Основано на Mask RCNN. 
![hippo](https://github.com/buzaev-fedor/Smart-Parking-System-MaskRCNN/blob/main/gif.gif)
## Принцип действия
1. Чертеж парковки - оператор расчерчивает парковочные места для парковки (с помощью set_regions.py). 
2. Для стабильной картинки в реальном времени, я делаю так: Берется один кадр, кадр обрабатывается и отдается в ОЗУ. Кадр показывается 6 раз, а затем берется следующий кадр.
3. Берется из потока фрейм, обрезается до площади парковки и отдается нейросети, которая на выход отдает bounding box объектов. (Специально только часть фрейма, потому что снижается нагрузка и меньше бесполезной информации).
4. Для того, чтобы расчитать правильность парковки, берутся площади автомобилей, а затем по метрике IoU по отношению к площади парковочного места.
